Recomendaciones de seguridad

Protección de archivos

Si se usa Rapid SCADA en un entorno corporativo, restrinja el acceso de los usuarios al directorio de instalación de Rapid SCADA, de forma predeterminada C:\SCADA\. Para hacerlo, abra las propiedades del directorio que contiene las aplicaciones Rapid SCADA, elija la pestaña Security y configure los derechos de acceso.

HTTPS

Configure un servidor web para habilitar el protocolo HTTPS para la aplicación Webstation. Con HTTPS, todo el tráfico entre un navegador y el servidor web, incluidas las contraseñas, está encriptado.

VPN

Use VPN para proporcionar acceso a usuarios externos. Si es posible, evite el acceso abierto a la Webstation desde el exterior.

Contraseñas

Cambie las contraseñas predeterminadas. Abra el proyecto con la aplicación Administrator, ingrese las nuevas contraseñas en la tabla Usuarios y actualice las contraseñas para conectarse al Server especificado en la configuración de la aplicación. Para crear contraseñas seguras, use un generador de contraseñas. Si una empresa usa Active Directory, la configuración de la autenticación en Rapid SCADA basada en Active Directory mejora la seguridad del sistema.
